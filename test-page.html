Teknoloji Tespiti

Bu sayfa aşağıdaki teknolojileri kullanmaktadır:

  • Bootstrap 5.1.3 (CSS Framework)
  • Font Awesome 6.0.0 (Icon Library)
  • Google Fonts (Web Fonts)

Güvenlik Özellikleri

Bu sayfa güvenlik testleri için:

  • HTTPS protokolü kullanıyor
  • Referrer Policy ayarlanmış
  • Meta generator tag'i var
  • Dış kaynaklardan CSS/JS yüklüyor

Test Senaryoları

  • Dış kaynaklardan CSS ve JavaScript yükleme
  • Form güvenliği testi
  • Meta tag analizi
  • Teknoloji fingerprinting
